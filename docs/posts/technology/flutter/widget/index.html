Expanded#

可以设置元素的占用空间比例大小。

const Expanded({
    Key key,
    int flex = 1,
    @required Widget child,
})
  • flex 占比大小，默认是100%
    • 多个Expanded并列时，单个Expanded占用空间为flex/sum(flex)
  • child 即需要分配的Widget
